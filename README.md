# wordleGame

# Get Started
Before starting you need to install psql on your device.


1. Clone the git repository
2. `cd frontend`
3. `npm install`
4. `cd ..\backend`
5. `npm install`
6. `cd ..`
7. `psql -U <username> -f database.psq`
8. enter the posgres password
9.  Create a .env
10. `pip install psycopg2`
11. `pip install dotenv`
12. `python filter.py`
